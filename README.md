## Featured Projects

### [Kerala Ayurveda RAG System](https://github.com/DeshDeepakKant/Kerala-Ayurveda-Agentic-AI)
> **A multi-layered RAG system for safe, authentic Ayurveda content generation.**
> Built a 6-agent orchestration framework using **LangGraph** to automate domain-specific content creation while enforcing strict medical authenticity.

- **Hybrid Search Architecture**: Implemented BM25 and Vector search (ChromaDB) for high-precision retrieval.
- **CRAG (Corrective RAG)**: Integrated a self-healing loop that evaluates retrieval quality and re-queries for missing context when needed.
- **Knowledge Graph Constraints**: Utilized Graph-based grounding to prevent hallucinations and ensure clinical alignment.
- `Python` · `LangGraph` · `ChromaDB` · `FastAPI` · `CRAG` · `Knowledge Graphs` · `Gemini`
